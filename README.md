## System Programming Team Project
2023년 2학기 시스템 프로그래밍 팀 프로젝트 : 스마트 통발
## 구현 기능들
### PI 1
- 주된 서버 역할
  -> 다른 파이들과의 연결을 담당
- 카메라 센서
  -> 물고기 판단 및 안드로이드에 실시간 영상 전달
- 모터 센서
  -> 미끼를 풀기 위한 용도
### PI 2
- LED 센서
  -> 어둡거나 유저가 원할때 on/off 가능
- 조도 센서
  -> LED를 자동으로 제어하기 위함
### PI 3
- 모터 센서 2개
  -> 통발의 앞 뒷문의 개폐를 담당
### PI 4
- 스피커
  -> 유저가 원할 때 통발의 찾기 위함
- GPS
  -> 유저가 요청했을 때 현재 위치를 전달
- 수질 센서
  -> 현재 통발 위치의 수질정보를 유저에게 보여줌
### ELSE
- flask&pytorch
  -> 물고기인지 쓰레기인지 판단
- android
  -> 유저의 통발 작동을 위함
